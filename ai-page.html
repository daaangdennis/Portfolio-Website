Feel free to draw on the canvas below!

This page was created with the help of ChatGPT in order to present the capabilities of Artificial Intelligence (AI).

The canvas and "Clear Canvas" button you see on the left were both generated completely by ChatGPT. Furthermore, the containers that are holding this textbox as well as the canvas were created with the help of ChatGPT.

Since drawing and creating art has always been one of my favorite pastimes, I wanted to use AI to help bring that experience to you on this webpage!

Here are some of the prompts that I used to create this page:

  • Can you make a canvas where the user can draw
  • Give the canvas a border color of #81a5f9
  • Put a text box to the right of the canvas
  • Give the text box the same border color as the canvas and make the corners rounded
  • Make the font in the text box "Roboto Slab"
  • Split the canvas and the text container into their own separate columns
  • Add a button that clears the canvas
  • Add text above the canvas that says "feel free to draw in the box below!"
