This page was created with the help of ChatGPT in order to present the capabilities of Artificial Intelligence (AI).

The canvas and "Clear Canvas" button you see on the left were both generated completely by ChatGPT. Furthermore, the containers that are holding this textbox as well as the canvas were created with the help of ChatGPT.

Since drawing has always been one of my favorite pastimes, I wanted to use AI to help bring that experience to you on this webpage! The following are some prompts that were used to create this page:

To create the navigation bar:

  • Start with a doctype declaration.
  • Create an HTML element with a head and body.
  • Inside the head, include a charset meta tag and a viewport meta tag.
  • Add a link element to include an external CSS file named "style.css."
  • Set the page title to "AI Page | Dennis Dang."
  • Include a favicon with the path "images/icon.ico."
  • Inside the body, create a div with a class of "navbar."
  • Inside the div, add an image with a class of "logoPic" and a source of "images/logo.svg."
  • Create a navigation menu using an unordered list (ul) with a class of "nav-links."
  • Add list items for each navigation link: About, Projects, Qualifications, Service, AI Page.
  • Include a "Contact" button that links to "mailto:dennisdang80@yahoo.com."

To create the text box and canvas:

  • Can you make a canvas where the user can draw
  • Give the canvas a border color of #81a5f9
  • The first div should have an id of "content-container."
  • Inside the "content-container" div, create another div with an id of "canvas-container."
  • Add an h2 element with the text "Feel free to draw on the canvas below!" within the "canvas-container" div.
  • Include a canvas element with an id of "drawing-canvas" and set its width and height attributes to 700.
  • Add a "Clear Canvas" button with an id of "clear-button."
  • The second div should have an id of "text-container."
  • Inside the "text-container" div, add paragraphs of text describing the purpose of the page.
  • Highlight one of the paragraphs with a style attribute setting the color to "#81a5f9."
  • List the prompts used to generate this page as list items within an unordered list.
  • Make the font in the text box "Roboto Slab"
  • Give the text box the same border color as the canvas and make the corners rounded
